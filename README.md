# suibi

肖恩聊技术个人门户。本站从 VuePress 迁移到 Astro，承载个人信息、博客归档、项目、工具入口、导航资源和推广推荐。

**访问站点**：[肖恩聊技术](https://shawnxie.top/)

![](home.png)

## 开发

```bash
npm install
npm run dev
npm run build
```

## 写新文章

使用脚本生成文章模板：

```bash
npm run new:post
npm run new:post -- "文章标题"
npm run new:post -- "文章标题" --open
```
